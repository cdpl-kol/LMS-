============================================================
   SMART LMS - BACKEND SETUP GUIDE
   Connecting Dot Pvt. Ltd.
============================================================

REQUIREMENTS:
  - Node.js (v16+) : https://nodejs.org
  - PostgreSQL      : https://www.postgresql.org/download/

------------------------------------------------------------
STEP 1: PostgreSQL Database Setup
------------------------------------------------------------
1. PostgreSQL install karo (agar nahi hai)
2. PostgreSQL open karo aur ek database banao:

   Windows (pgAdmin use karo) ya Command Prompt mein:
   > psql -U postgres
   > CREATE DATABASE lms_db;
   > \q

------------------------------------------------------------
STEP 2: .env file configure karo
------------------------------------------------------------
backend/.env file mein apna PostgreSQL password daalo:

   DATABASE_URL=postgresql://postgres:YOUR_PASSWORD@localhost:5432/lms_db

   Example: agar password "1234" hai:
   DATABASE_URL=postgresql://postgres:1234@localhost:5432/lms_db

------------------------------------------------------------
STEP 3: Dependencies install karo
------------------------------------------------------------
Command Prompt / Terminal mein:

   cd LMS/backend
   npm install

------------------------------------------------------------
STEP 4: Server start karo
------------------------------------------------------------
   node server.js

   Aapko yeh message aayega:
   ✅ Database initialized successfully
   🚀 LMS Backend running at: http://localhost:3001

------------------------------------------------------------
STEP 5: Browser mein kholो
------------------------------------------------------------
   http://localhost:3001

DATA SAVE HONE KA PROCESS
------------------------------------------------------------

    mein save hoga permanently
