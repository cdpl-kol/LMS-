============================================================
   SMART LMS - BACKEND SETUP GUIDE
   Connecting Dot Pvt. Ltd.
============================================================

REQUIREMENTS:
  - Node.js (v16+) : https://nodejs.org
  - PostgreSQL      : https://www.postgresql.org/download/

------------------------------------------------------------
STEP 1: PostgreSQL Database Setup
------------------------------------------------------------
1. Install PostgreSQL (if not already installed)
2. Open PostgreSQL and create a database:

   Windows (use pgAdmin) or Command Prompt:
   > psql -U postgres
   > CREATE DATABASE lms_db;
   > \q

------------------------------------------------------------
STEP 2: Configure the .env file
------------------------------------------------------------
Add your PostgreSQL password in the backend/.env file:

   DATABASE_URL=postgresql://postgres:YOUR_PASSWORD@localhost:5432/lms_db

   Example: if your password is "1234":
   DATABASE_URL=postgresql://postgres:1234@localhost:5432/lms_db

------------------------------------------------------------
STEP 3: Install Dependencies
------------------------------------------------------------
In Command Prompt / Terminal:

   cd LMS/backend
   npm install

------------------------------------------------------------
STEP 4: Start the Server
------------------------------------------------------------
   node server.js

   You should see this message:
   ✅ Database initialized successfully
   🚀 LMS Backend running at: http://localhost:3001

------------------------------------------------------------
STEP 5: Open in Browser
------------------------------------------------------------
   http://localhost:3001

------------------------------------------------------------
HOW DATA IS SAVED
------------------------------------------------------------
  - Any add/edit/delete action is saved permanently in the PostgreSQL database
  - You can stop and restart the server — data will remain safe
  - Data is only lost if explicitly deleted

------------------------------------------------------------
STOPPING THE BACKEND
------------------------------------------------------------
  Press Ctrl+C in the terminal
